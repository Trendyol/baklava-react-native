# Sizes

Baklava uses a list of defined size for customizing the global spacing, font sizes, font weights, radius

## Space

```jsx
const spacing = [-80, -64, -48, -40, -32, -24, -20 -16, -14, -12, -8, -4, -2, 
                 0, 2, 4, 8, 12, 14, 16, 20, 24, 32, 40, 48, 64, 80];
```

## Radius

```jsx
export const borderRadii = {
  none: 0,
  xs: 2,
  s: 4,
  m: 6,
  l: 8,
  xl: 16,
  full: 999,
};
```
