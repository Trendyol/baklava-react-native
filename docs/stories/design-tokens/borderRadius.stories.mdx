# Border Radius

The border-radius property defines the radius of the element's corners.

```jsx
export const borderRadii = {
  none: 0,
  xs: 2,
  s: 4,
  m: 6,
  l: 8,
  pill: 999,
};
```

### Usage

```jsx

```
