## Usage

```jsx
<Switch
    disabled={true}
    checked={false}
    customization={{colorOn: '#0BC15C', colorOff: '#FF5043'}}
    onChange={(value) => {console.log(value)}}
/>
```

<br />

## Props

### onChange

Type: `() => void`

Callback fired when the switch is checked

### label

Type: `string`

Optional label for the switch

### disabled

Type: `boolean`

Optional prop to make the component disabled

### checked

Type: `boolean`

Optional prop to make the component checked by default

#### onColor

Type: `string`

The on color of the Switch. 
Default value is `primaryKey`. 
It can be <a href="https://reactnative.dev/docs/colors#named-colors" title="title">ColorValue</a>.

#### offColor

Type: `string`

The off color of the Switch. 
Default value is `neutralLighter`. 
It can be <a href="https://reactnative.dev/docs/colors#named-colors" title="title">ColorValue</a>.
