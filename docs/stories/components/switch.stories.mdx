## Usage

```jsx
<Switch
    label="Notifications"
    value={false}
    onValueChange={(value) => {console.log(value)}}
    disabled={false}
    onColor="#0BC15C"
    offColor="#FF5043"
/>
```

<br />

## Props

#### onValueChange

Type: `(value: boolean) => void`

Callback fired when the switch value changes. Receives the new boolean value as a parameter.

#### label

Type: `string`

Optional label displayed next to the switch.

#### value

Type: `boolean`

Optional prop to set the initial state of the switch (on/off).

#### disabled

Type: `boolean`

Optional prop to make the component disabled. When disabled, opacity is reduced to 0.5.

#### onColor

Type: `string`

The background color of the switch when it is in the "on" state.
Default value is `primaryKey`.
It can be <a href="https://reactnative.dev/docs/colors#named-colors" title="title">ColorValue</a>.

#### offColor

Type: `string`

The background color of the switch when it is in the "off" state.
Default value is `neutralLighter`.
It can be <a href="https://reactnative.dev/docs/colors#named-colors" title="title">ColorValue</a>.

#### containerStyle

Type: `StyleProp<ViewStyle>`

Optional style prop applied to the outer container of the switch.

#### testID

Type: `string`

Optional test identifier for the component.

#### accessibilityLabel

Type: `string`

Optional accessibility label for screen readers.
