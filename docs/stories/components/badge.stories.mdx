## Usage

```jsx
  <Badge variant="success" text="Badge Component" />
```

## Props

#### variant
  
Type: `string`

The variant of the badge style to use.

`default`,
`neutral`,
`success`,
`warning`,
`danger`

#### text

Type: `string`

Badge text.

#### icon

Type: `string`

Badge icon.

#### transparent

Type: `boolean`

Badge without backgroundColor.

#### backgroundColor

Type: `string`

Pass the transparent prop to give a custom backgroundColor.

#### style

Type: `StyleProp<ViewStyle>`
