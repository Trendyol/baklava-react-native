## Usage

```jsx
<FlagIcon name="TR" size="xl" />
```

<br />

## Props

#### name

Type: `string`

Flag Icon names are in the list, fixed and cannot be used outside the list.

`CZ`,
`DE`,
`GR`,
`QA`,
`RO`,
`SA`,
`SK`,
`TR`,
`AE`,

#### size

Type: `string`

The size of the flag icon.

`xl`,
`l`,
`m`,
`s`,
`xs`,
`2xs`
