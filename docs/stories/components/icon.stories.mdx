## Usage

```jsx
<Icon name="confetti" color="primaryColor" size="xlarge" />
```

<br />

## Props

#### name

Icon names are in the list, fixed and cannot be used outside the list.

#### size

The size of the icon.

`xlarge`,
`large`,
`medium`,
`small`
`xsmall`
`2xsmall`
