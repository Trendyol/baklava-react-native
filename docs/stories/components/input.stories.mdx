## Usage

```jsx
<Input size="large" placeholder="Large" />
```

## Props

#### size

The size of the button.
`large`,
`medium`,
`small`

#### label

The text or component to use for the floating label.

#### labelFixed

Boolean value that checks if the label is fixed.

#### placeholder

The string that will be rendered before text input has been entered.

#### helpText

Helper text providing information for the input.

#### errorText

Helper text showing the error in the input.

#### successText

Helper text showing the success in the input.

#### icon

The icon shown in the input.

#### success

Boolean value indicating the success status.

#### error

Boolean value indicating the error status.

#### secureTextEntry

If true, the text input obscures the text entered so that sensitive text like passwords stay secure. The default value is false.

#### required

Boolean value that controls whether the field is required.

#### disabled

If true, user won't be able to interact with the component.

#### style

Type: `StyleProp<TextInputStyle>`
