## Usage

```jsx
<Button variant="primary" type="outline" size="large" text="Primary Button" />
```

<br />

## Props

#### variant

The variant of the button style to use.

`primary`,
`secondary`,
`success`,
`danger`,
`passive`

#### type

The type of the button. To adjust the styling to give it desired emphasis.

`contained`,
`outline`,
`text`

#### size

The size of the button.

`large`,
`medium`,
`small`

#### style

Type: `StyleProp<PressableStyle>`
