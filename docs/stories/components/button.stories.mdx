## Usage

```jsx
<Button variant="primary" kind="success" size="l" label="Primary Button" />
```

<br/>

## Props

#### variant

Type: `string`

The variant of the button style to use.

`primary`,
`secondary`,
`tertiary`

#### kind

Type: `string`

The kind of the button. To adjust the styling to give it desired emphasis.

`default`,
`neutral`,
`success`,
`danger`

#### size

Type: `string`

The size of the button.

`s`,
`m`,
`l`

#### icon

Type: `string`

The icon name of the button. Icon names are in the list, fixed and cannot be used outside the list.

`academy`,
`confetti`,
`info`,
`profile`,
`warning`...

#### label

Type: `string`

The label of the button.

#### disabled

Type: `boolean`

Disabled state of the button. Default value is false.

#### loading

Type: `boolean`

Loading state of the button. When true, displays a spinner instead of the icon and automatically disables the button. Default value is false.

#### filled

Type: `boolean`

Filled state of the button. Default value is false.

#### style

Type: `StyleProp<PressableStyle>`
