## Props

#### variant

The variant of the button style to use.

`primary`,
`secondary`,
`tertiary`,

#### kind

The kind of the button. To adjust the styling to give it desired emphasis.

`default`,
`neutral`,
`success`
`danger`

#### size

The size of the button.

`s`,
`m`,
`l`

#### style

Type: `StyleProp<PressableStyle>`
