# Text

Text allows the rendering of text and paragraphs within an interface. Shows styles compatible with the passed variant prop and supported by the type system.

## Props

#### variant

Variant defines appropriate text styles for type role and its size. Available variants:

Heading: _`heading1`, `heading2`, `heading3`_ <br />
Subtitle01: _`subtitle01Regular`, `subtitle01Medium`, `subtitle01Semibold`, `subtitle01Bold`_ <br />
Subtitle02: _`subtitle02Regular`, `subtitle02Medium`, `subtitle02Semibold`, `subtitle02Bold`_ <br />
Subtitle03: _`subtitle03Regular`, `subtitle03Medium`, `subtitle03Semibold`, `subtitle03Bold`_ <br />
Subtitle04: _`subtitle04Regular`, `subtitle04Medium`, `subtitle04Semibold`, `subtitle04Bold`_ <br />
Body: _`bodyText`, `bodyUnderline`, `bodyTextLink`, `bodyLongText`_ <br />
Caption: _`captionText`, `captionMedium`, `captionLongText`, `captionTextLink`_ <br />

#### children (required)

Type: `React.ReactNode`

#### theme

Type: `ThemeProp`

#### style

Type: `StyleProp<TextStyle>`
