## Usage

```jsx
<RadioButton disable={false} selected={true} label="lorem ipsum" flagIcon="TR" />
```

<br />

## Props

### selected

Type: `boolean`

Use selected state when the item is selected.

### label

Type: `string`

The label defines a caption for an text.

### flagIcon

Type: `string`

Radio Button flag icon.

### disabled

Type: `boolean`

If true, user won't be able to interact with the component.

#### style

Type: `StyleProp<TouchableOpacityStyle>`
