## Usage

```jsx
<Alert caption="caption" description="description" closable={true}>
  <Button label="Action Button" />
</Alert>
```

<br />

## Props

#### variant

Type: `string`

The variant of the alert style to use.

`info`,
`warning`,
`success`,
`danger`,

#### closable

Type: `boolean`

Displays a close button.

#### caption

Type: `string`

Alert caption.

#### description

Type: `string`

Alert description.

#### closed

Type: `boolean`

Alert description

#### onClose

Type: `() => void`

Callback fired when the component requests to be closed.

#### children

Type: `React.ReactNode`

#### style

Type: `StyleProp<ViewStyle>`
