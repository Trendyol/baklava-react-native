## Usage

```jsx
<Checkbox disable={false} checked={true} indeterminate={true} label="lorem ipsum" flagIcon="turkey" />
```

<br />

## Props

### checked

Type: `boolean`

Use selected state when the item is selected.

### indeterminate

Type: `boolean`

Use the indeterminate state when the checkbox contains a sublist of selections, some of which are selected, and some unselected.

### label

Type: `string`

The label defines a caption for an text.

### flagIcon

Type: `string`

Checkbox flag icon.

### disabled

Type: `boolean`

If true, user won't be able to interact with the component.
