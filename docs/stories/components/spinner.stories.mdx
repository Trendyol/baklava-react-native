## Usage

```jsx
<Spinner color="primaryColor" size="xl" />
```

<br />

## Props

#### size

The size of the spinner.
Default value is `m`

`2xl`,
`xl`,
`l`,
`m`,
`s`,
`xs`,
`2xs`

#### color

The color of the spinner.
Default value is `primaryKey`. It can be <a href="/docs/design-system-colors--page" title="title">theme color key</a> or <a href="https://reactnative.dev/docs/colors#named-colors" title="title">ColorValue</a>
  
ex: 
`neutralDarker`,
`red`,
`maroon`,
`#EBEF95`,
`#BEADFA`
