Text link is a navigational element that takes a user to another URL, an element within a page, or a file.

## Usage

```jsx
const [isPressed, setIsPressed] = useState(false)
// ...
<TextLink 
  variant="subtitle3Medium" 
  onPress={() => setIsPressed(true)} 
  isPressed={isPressed}
>
  Example
</TextLink>
```

<br />

## Props

#### variant

Variant defines appropriate text styles for type role and its size. Available variants:

Heading: _`heading1`, `heading2`, `heading3`_ <br />
Subtitle1: _`subtitle1Regular`, `subtitle1Medium`, `subtitle1Semibold`, `subtitle1Bold`_ <br />
Subtitle2: _`subtitle2Regular`, `subtitle2Medium`, `subtitle2Semibold`, `subtitle2Bold`_ <br />
Subtitle3: _`subtitle3Regular`, `subtitle3Medium`, `subtitle3Semibold`, `subtitle3Bold`_ <br />
Subtitle4: _`subtitle4Regular`, `subtitle4Medium`, `subtitle4Semibold`, `subtitle4Bold`_ <br />
Body: _`body1`, `body2`, `body3`_ <br />
Caption: _`caption`_ <br />

#### children (required)

Type: `React.ReactNode`

#### isPressed

Type: `boolean`

#### onPress

Type: `function`

#### theme

Type: `ThemeProp`

#### style

Type: `StyleProp<TextStyle>`
