# Box

This is a generic component for low level layout needs. It is similar to a `div` in HTML and `View` in React Native.

<div
  style={{
    display: 'flex',
    flexWrap: 'wrap',
    justifyContent: 'center',
  }}>
  <img src={Box} width="250" alt="Box" />
</div>

<Box bg="primaryColor" />
