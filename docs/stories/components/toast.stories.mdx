## Usage
Add it to root view so that it can be seen above all views.
```jsx
// ...
import { theme, ThemeProvider, Toast } from '@trendyol/baklava-react-native';
// ...

<ThemeProvider theme={theme}>
  <SafeAreaProvider>
    {/* other components or providers. ex: <App/> */}
    <Toast ignoreKeyboard extraPaddingBottom={40}/>
  </SafeAreaProvider>
</ThemeProvider>

// ...
```

<br />

## Props

#### variant

Type: `string`

The variant of the toast style to use.

`default`,
`success`,
`error`,
`warning`,

#### ignoreKeyboard

Type: `boolean`

It is used to position according to the visibility of the keyboard.

#### extraPaddingBottom

Type: `number`

It is used to position according to components such as bottom tabbar.


## Methods

#### show
```jsx
// without action
Toast.show({
  text: 'Lorem ipsum dolor sit amet',
})
```

```jsx
// with action
Toast.show({
  variant: 'error',
  action: () => { /* do something */},
  actionText: 'Action',
  duration: ToastDurations.LONG,
  text: 'Lorem ipsum dolor sit amet.',
  bottomOffset: 30
})
```
